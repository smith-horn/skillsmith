# Skillsmith Logo Concepts

**Purpose:** Logo exploration for Skillsmith brand identity
**Direction:** Bold & Confident, Claude-aligned warmth, developer-focused

---

## Brand Context for Logo

**What Skillsmith does:** Intelligent skill discovery for Claude Code
**Key metaphors:**
- Finding/discovering (search, lens, compass)
- Skills/abilities (tools, hands, craft)
- Layering/stacking (cards, layers, building)
- Smithing/forging (anvil, hammer, craft)
- Intelligence/AI (neural, connections, spark)

**Personality:** Confident, technical, warm, efficient

## Concept 1: The Skill Stack

### Description
Abstract representation of layered skill cards, suggesting depth of the ecosystem and the discovery/selection process.

```
Visual:
┌─────────┐
│  ┌───┐  │    Three overlapping rectangles
│  │ ┌─┼──┼─┐  Offset diagonally (top-right)
│  │ │ │  │ │  Creates depth and movement
└──┼─┼─┘  │ │  Suggests browsing through skills
   │ └────┼─┘
   └──────┘

Refined version:
    ╱─────╲
   ╱ ╱───╲ ╲     Rounded corners
  ╱ ╱     ╲ ╲    Coral gradient on front card
 ╱_╱_______╲_╲   Gray/muted on back cards
```

### Rationale
- Directly represents "skills" as discoverable units
- Stacking implies the vast ecosystem (46K+)
- Front card prominence = finding THE right skill
- Clean, geometric, scales well

### Color Application
- Front card: Coral `#E07A5F`
- Middle card: `#A1A1AA` (40% opacity)
- Back card: `#71717A` (20% opacity)

### Wordmark Pairing
```
[Stack Icon] skillsmith
             ^^^^^^^^^^
             Satoshi Bold, lowercase
```

```
Visual:

   │           │     Handle extends bottom-right
    ╲         ╱      S is subtle, negative space
     ╰───────╯
           ╲
            ╲

Minimal version:
     ○───        Simple circle with offset line
                 S implied by the shape itself
```

### Rationale
- Universal "search" symbol
- The "S" creates brand recognition
- Lens = focus, clarity, finding what matters
- Works at small sizes (favicon)

### Color Application
- Circle stroke: Coral `#E07A5F`
- Handle: Coral or white
- S interior: Negative space (background shows through)

### Wordmark Pairing

```
  ●       ●       Nodes at key points
   ╲     ╱        Lines connecting nodes
    ●───●         Neural/network aesthetic
   ╱     ╲
```

## Concept 5: The Bracket Frame

### Description
Code brackets `< >` or `{ }` framing an abstract element, directly referencing the developer context.

```
Visual:
   <  ◆  >        Angle brackets with diamond

   <  S  >        Angle brackets with S

   {  ●  }        Curly braces with dot

   < ─── >        Brackets with line (minimal)

Refined:
   ╱        ╲
  ╱    ◇    ╲     Stylized brackets
  ╲          ╱    Diamond center (skill)
   ╲        ╱
```

### Rationale
- Instant developer recognition
- Brackets = code, syntax, technical
- Center element = the skill being discovered
- Versatile, scales well, memorable

### Color Application
- Brackets: White or light gray
- Center element: Coral `#E07A5F`
- Creates natural focal point on accent

### Wordmark Pairing
```
< ◆ > skillsmith
      ^^^^^^^^^^
      JetBrains Mono, lowercase (code aesthetic)

or

<skillsmith>
^^^^^^^^^^^^
Brackets as part of wordmark
```

## Concept 6: The Command Prompt

### Description
Terminal prompt symbol (❯ or >) stylized as a distinctive mark, embracing the CLI-native positioning.

```
Visual:
    ❯              Single chevron, bold weight

    ▶              Filled triangle (play/execute)

   ❯❯              Double chevron (forward motion)

   ❯_              Chevron with cursor underscore

Refined:
    ╱
   ╱               Bold chevron
  ╱                Slight coral gradient
 ╱                 Suggests forward movement
```

### Rationale
- Terminal-native product, terminal-native logo
- Universal "prompt" recognition among developers
- Implies action, execution, moving forward
- Extremely simple, highly memorable

### Color Application
- Chevron: Coral `#E07A5F`
- Solid fill or gradient (top to bottom)
- Cursor underscore: White (if included)

### Wordmark Pairing
```
❯ skillsmith
  ^^^^^^^^^^
  Satoshi Bold or JetBrains Mono

or

❯ SKILL
  SMITH          Stacked wordmark
```

## Recommended Direction

Based on the brand positioning (Bold & Confident, Claude-aligned, Optimizer persona), I recommend exploring these three concepts further:

### Primary Recommendation: Concept 1 (Skill Stack)
**Why:** Directly represents the product's core value, distinctive, works at all sizes, feels premium and modern.

### Secondary: Concept 6 (Command Prompt)
**Why:** Maximum simplicity, instant developer recognition, highly memorable, terminal-native authenticity.

### Tertiary: Concept 4 (Neural S)
**Why:** Strong brand letterform, AI/intelligence association, Claude ecosystem alignment.

## Wordmark Typography Options

### Option A: Satoshi Bold, Lowercase
```
skillsmith
```
Modern, approachable, confident. Lowercase feels technical.

### Option B: Satoshi Black, Title Case
```
Skillsmith
```
More formal, stronger presence, clear brand name.

### Option C: Satoshi Bold, Split
```
skill smith
```
Emphasizes both words, reads as two concepts combined.

### Option D: JetBrains Mono, Lowercase
```
skillsmith
```
Maximum technical credibility, matches terminal aesthetic.

### Option E: Mixed Weight
```
SKILLsmith
```
or
```
skillSMITH
```
Creates visual interest, emphasizes one concept.

**Recommendation:** Option A (Satoshi Bold, lowercase) for approachability, or Option D (JetBrains Mono) for Variant B terminal aesthetic.
