## Priority Goals by Journey Stage

| Journey Stage | Primary Goal | Secondary Goal | Tertiary Goal |
|---------------|--------------|----------------|---------------|
| **First Discovery** |
| Awareness | Understand value | Assess effort | Verify trust |
| Trial | Test safely | Get quick value | Maintain control |
| First Value | Experience benefit | Confirm expectations | Plan next steps |
| Return | Decide commitment | Identify more value | Share experience |
| **Daily Workflow** |
| Trigger | Recognize opportunity | Minimal disruption | Quick access |
| Engagement | Find relevant info | Fast response | Clear presentation |
| Value | Solve problem | Verify help | Note for future |
| Exit | Return to work | Clean exit | No lingering |
| **Team Adoption** |
| Champion | Validate for team | Prepare justification | Initiate discussion |
| Trial | Test with team | Monitor adoption | Collect feedback |
| Standardization | Consistent setup | Onboarding process | Policy enforcement |
| Scale | Fast onboarding | Standard maintenance | Exception handling |
| **Skill Creation** |
| Create | Improve workflow | Build effectively | Test thoroughly |
| Distribute | Publish easily | Understand quality | Get feedback |
| Discovery | Get found | Monitor installs | Respond to users |
| Reputation | Build credibility | Maintain skill | Grow portfolio |
