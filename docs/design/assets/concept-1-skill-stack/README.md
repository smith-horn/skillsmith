# Concept 1: Skill Stack

Three overlapping rounded rectangles arranged diagonally, representing discoverable skills.

## Visual
```
     ┌────────┐
    ┌┼────────┼┐
   ┌┼┼        ┼┼┐
   │││  ████  │││
   └┼┼        ┼┼┘
    └┼────────┼┘
     └────────┘
```

## Rationale
- Directly represents "skills" as discoverable units
- Stacking implies the vast ecosystem (46K+)
- Front card prominence = finding THE right skill
- Clean, geometric, scales well

## Colors
- Front card: Coral `#E07A5F`
- Middle card: White at 30% opacity
- Back card: White at 15% opacity
- Background: `#0D0D0F`
