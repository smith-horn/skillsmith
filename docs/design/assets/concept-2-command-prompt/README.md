# Concept 2: Command Prompt

Bold chevron (>) representing the terminal prompt — maximum developer recognition.

## Visual
```
    ╱
   ╱
  ╱      >
 ╱
╱
```

## Rationale
- Terminal-native product, terminal-native logo
- Universal "prompt" recognition among developers
- Implies action, execution, moving forward
- Extremely simple, highly memorable
- Maximum technical credibility

## Colors
- Chevron: Coral gradient `#E07A5F` → `#D4694E`
- Background: `#0D0D0F`
- Subtle glow effect on refined version
